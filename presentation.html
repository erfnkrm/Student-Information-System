Docker Configuration

Multi-Stage Dockerfile

FROM ubuntu:22.04 AS build
# Build stage with CMake and dependencies
RUN apt-get install build-essential cmake libpqxx-dev
COPY . .
RUN cmake -B build && cmake --build build

FROM ubuntu:22.04 AS runtime
# Lightweight runtime stage
COPY --from=build /app/build/sis /app/sis
ENTRYPOINT ["/app/sis"]

Benefits: Smaller image size, faster deployment, separation of build and runtime dependencies

Quick Start Guide

Starting the System

# Clone the repository
git clone https://github.com/erfnkrm/cpp-finalproject-ERS
cd cpp-finalproject-ERS

# Start the database
docker compose up db -d

# Run the application
docker compose run --rm app

Stopping the System

# Stop all services
docker compose down

# Remove volumes (reset database)
docker compose down -v
